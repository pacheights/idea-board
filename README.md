# Idea Book![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)

Idea Book is a website that keeps track of all of my app/business/music/film ideas

## Installation

### Requirements
* Node.js
* Node live-server
* Google Chrome

## Usage

```javascript
npm install

npm start
```
